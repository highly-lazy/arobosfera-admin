# Arabosfera Admin

Duolingo uslubidagi, to‘liq o‘zbek tilidagi React admin panel.

## Asosiy modullar

- Admin login
- Gamified dashboard
- Foydalanuvchilar, XP, progress va obunalarni boshqarish
- Testlar CRUD
- Mammoth.js asosidagi DOCX test parser
- Imtihon natijalari va Writing/Speaking baholash
- Oktagon jonli monitoring simulyatsiyasi
- Analitika va diagrammalar
- Bildirishnomalar
- Promo-kodlar
- Murojaatlar
- Tizim loglari
- Audio fayllar ombori
- Theme, taymer, AI baholash va imtihon sana sozlamalari

## Ishga tushirish

```bash
npm install
npm run dev
```

Brauzerda: `http://localhost:5173`

## Demo login

- Login: `admin@arabosfera.uz`
- Parol: `admin123`

Hozirgi versiya mock data va localStorage bilan ishlaydi. Haqiqiy backend kelganda `src/data/mock.ts` o‘rniga API service qatlamini ulash mumkin.
